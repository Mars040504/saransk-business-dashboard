# 📊 Ключевые бизнес-метрики в Саранске (май – июнь 2021)

## Описание проекта

Интерактивный дашборд для анализа ключевых бизнес-метрик сервиса доставки еды в городе Саранск.

## Цель проекта

Проанализировать ключевые показатели пользовательской активности, удержания и эффективности бизнеса, выявить закономерности и подготовить рекомендации.

## Что реализовано

- KPI (DAU, Conversion Rate, Retention Rate, LTV, средний чек)
- Анализ динамики пользовательской активности
- Анализ удержания пользователей
- Анализ LTV ресторанов и блюд
- Аналитический отчет с выводами и рекомендациями

## Инструменты

- Yandex DataLens
